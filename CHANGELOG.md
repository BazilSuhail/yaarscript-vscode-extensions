# Change Log

All notable changes to the "yaarscript" extension will be documented in this file.

## [1.0.0] - Initial Release
- **Syntax Highlighting**: Complete grammar mappings for all YaarScript keywords and operators.
- **Code Runner**: Integrated a one-click Run button on the editor title menu to execute code instantly.
- **Terminal Execution**: Supports interactive command-line inputs smoothly inside VS Code's integrated terminal.
- **Cross-Platform Engine**: Bundled native `yaar-windows.exe`, `yaar-macos`, and `yaar-linux` executables that auto-switch depending on the OS platform.
- **Theming**: Included the "YaarScript Dark" custom UI theme specifically colored for the language tokens.
- **Marketplace Readiness**: Added repository metadata, author information, and marketplace-ready UI graphics.
