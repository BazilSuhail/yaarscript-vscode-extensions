## Overview

YaarScript Snippets Pack is a professional-grade template library designed for developers who want to write YaarScript logic with speed and precision. This extension provides all the mandatory boilerplates, common algorithms, and syntax patterns through simple **`yr-`** prefixes and tab-friendly logic.

Whether you're starting a new program with the required `yaar {}` entry point or generating complex numeric graphics, this pack eliminates the need to remember every localized token manually.

---

## How to Use

1.  Open or create any file with the **`.yaar`** extension.
2.  Start typing a snippet prefix (e.g., `yr-init` or `yr-yaar`).
3.  When the suggestion list appears, press **Enter** or **Tab** to inject the template.
4.  Use the **Tab** key to jump between custom fields (like variable names and loop limits).

## Comprehensive Snippet Catalog

Below is the complete list of templates included in this pack, categorized for your workflow.

### Project Starters
- **`yr-init` (Full Project Template)**: The professional way to start! Generates a complete file structure including a header, global constants (`pakka`), helper function routines, and the mandatory `yaar {}` entry point.
- **`yr-yaar` (Mandatory Entry Point)**: Generates the `yaar {}` block required for any executable YaarScript program.

### Core Component Snippets
- **`yr-agar` / `yr-warna`**: Standard If and If-Else branching with instructional comments on boolean conditions.
- **`yr-intekhab` (Case Switch)**: A full selection statement with a helpful reminder to declare the switch variable first!
- **`yr-dohrao` (For Loop)**: Generates a typical index-based iteration pattern.
- **`yr-jabtak` (While Loop)**: Generates a condition-based iteration loop.
- **`yr-karo` (Do-While)**: Generates a loop that executes at least once.
- **`yr-qism` (Enum/Type)**: Easily define custom data models and enumerations.
- **`yr-khaali` (Void Routine)**: Scaffold for functions that do not return a value.
- **`yr-func` (Typed Function)**: Scaffold for functions returning `number`, `float`, or `faisla` types via `wapsi`.

### Data Declaration and I/O
- **`yr-pakka` (Immutable Constant)**: Safeguard your logic with variables that cannot change after initialization.
- **`yr-var` (Mutable Variable)**: Standard variable declaration with type-selection.
- **`yr-bolo` (Print Statement)**: High-speed terminal output template.
- **`yr-suno` (Capture Input)**: Logic to capture user input into variables.

### Advanced Code Demos
- **`yr-diamond` (Graphics Logic)**: A complex nested-loop generator that creates a numeric diamond pattern—perfect for checking your language results!
- **`yr-sensor` (Sensor Calibration Demo)**: A pre-built "interactive" loop demonstrating automated attempts and status checking.
- **`yr-timer` (Performance Benchmarks)**: Automatically wraps your logic with `waqt()` logic to calculate execution time.
- **`yr-random` (Probability Set)**: High-speed logic for range-based random number results via `ittifaq()`.

## About YaarScript

YaarScript is an educational programming language meant to bridge the gap between Urdu speakers and computer science fundamentals. This snippet pack ensures your focus remains on logic and algorithms rather than typing manual syntax.
